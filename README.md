# 🏥 Clinic Appointment Booking System

A simple, responsive web app for managing clinic appointments. Book, view, and cancel appointments with automatic double-booking prevention. Built with HTML, CSS, and Vanilla JavaScript.

## ✨ Quick Features

- 📅 **Easy Booking** - Simple form with real-time validation
- 🔒 **No Double Bookings** - Disabled slots prevent conflicts
- 👥 **5 Doctors** - Select from different specialists
- 🗓️ **Smart Filtering** - Filter by date or doctor
- 📱 **Fully Responsive** - Works on all devices
- 💾 **Auto-Save** - Data persists with localStorage
- 🎨 **Modern Design** - Professional healthcare theme

## 🛠️ Technologies

- HTML5, CSS3, Vanilla JavaScript
- Font Awesome Icons
- localStorage for data persistence

## 📁 Files

- `index.html` - Main page
- `style.css` - Styling
- `script.js` - Functionality

## 🎯 Features Explained

- **Auto-Save** - Your appointments save automatically in browser
- **Double-Booking Prevention** - Booked time slots are automatically disabled
- **Responsive** - Works perfectly on desktop, tablet, and mobile
- **Smooth UI** - Fast, no page reloads needed
